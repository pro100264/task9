CCS3

Unlike CSS 2, which is a large single specification defining various features,
CSS3
is divided into several separate documents called "modules". Each module adds new capabilities or extends features defined in CSS 2, preserving backward compatibility. Work on CSS level 3 started around the time of publication of the original CSS 2 recommendation.

Due to the modularization,
different modules
have different stability and statuses. As of June 2012, there are over fifty CSS modules published from the CSS Working Group., and four of these have been published as formal recommendations.

Some modules have
Candidate Recommendation (CR)
status and are considered moderately stable. At CR stage, implementations are advised to drop vendor prefixes.
